# 📊 Dashboard de Vendas — Análise Estratégica com Tableau

Este projeto apresenta um **dashboard interativo de análise de vendas**, desenvolvido no Tableau, com foco na geração de **insights estratégicos a partir de dados**.

A análise explora padrões de desempenho **geral e regional**, incluindo faturamento, lucro, comportamento do cliente e sazonalidade.

---

## 🎯 Objetivo

- Analisar o desempenho geral de vendas  
- Identificar variações regionais  
- Avaliar lucro vs receita  
- Explorar comportamento de clientes  
- Apoiar decisões orientadas por dados
